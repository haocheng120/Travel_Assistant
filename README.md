# Travel Assistant

這是一個專為旅行設計的輕量級、純前端網頁應用程式。結合了「行程規劃」、「旅費記帳」與「購物清單」三大核心功能。透過 PWA (Progressive Web App) 技術，使用者可以直接將網頁加入手機主畫面，享受如同原生 App 般的操作體驗。

## ✨ 核心功能 (Features)

### 1. 🗓️ 互動式行程管理 (Itinerary)
* **直覺操作**：支援長按拖曳排序行程順序，向右滑動修改、向左滑動刪除。
* **動態交通導航**：內建「大眾運輸、步行、汽車」切換按鈕，點擊後自動生成 Google Maps 導航連結（自動帶入起迄點）。
* **跨平台地圖支援**：一鍵跳轉 Naver Map 查看精確的韓國在地圖資與店家資訊。
* **航班連動**：去回程航班資訊即時查詢（整合 FlightAware 動態）。

### 2. 💸 旅費記帳 (Expense Tracking)
* **雙幣別換算**：輸入匯率後，支援 TWD / KRW 雙幣別即時切換與加總。
* **預算進度條**：視覺化顯示剩餘預算百分比，避免超支。
* **退稅篩選器**：透過機票風格的標籤，一鍵過濾出「已退稅 / 未退稅」的消費明細，方便機場統一辦理退稅手續。
* **手勢操作**：記帳明細同樣支援側滑編輯與刪除。

### 3. 🛍️ 購物清單與行前準備 (Shopping & Checklist)
* **分類管理**：內建藥局、明洞、弘大、聖水洞、超市等自訂分類清單。
* **圖片附註**：支援上傳參考圖片與詳細備註（如色號、尺寸），採買不買錯。
* **行前檢查表**：護照、K-ETA、 Wowpass 等重要事項勾選確認。

---

## 🚀 系統架構與技術技術 (Tech Stack)

* **前端介面**：純 HTML, CSS (自訂變數與 Flexbox 排版), 原生 Vanilla JavaScript。
* **資料儲存**：使用瀏覽器 `localStorage` 進行本地端資料持久化，無須部署後端資料庫，保有最高隱私。
* **拖曳套件**：整合 `Sortable.js` 實現流暢的拖曳排序體驗。
* **深色模式**：支援系統級 Dark Mode 自動偵測與手動切換。

## 📱 安裝與使用方式 (Installation & Usage)

由於本專案支援 PWA，強烈建議在手機端以 App 形式安裝以獲得最佳體驗：

### iOS (Safari)
1. 使用 Safari 瀏覽器開啟本網頁。
2. 點擊下方工具列的「分享」圖示 (方塊往上加一個箭頭)。
3. 選擇 **「加入主畫面」 (Add to Home Screen)**。
4. 在主畫面點擊 App 圖示即可開始使用。

### Android (Chrome)
1. 使用 Chrome 瀏覽器開啟本網頁。
2. 點擊右上角「三個點」選單。
3. 選擇 **「加到主畫面」 (Add to Home Screen)**。
4. 授權後即可在桌面看到 App 圖示。

*(註：若要重置所有資料，可至「首頁 > 旅行基礎設定」最下方點擊清除資料按鈕。)*
